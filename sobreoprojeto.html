Ideia do Projeto

A ideia para a criação do jogo e do site com o assunto de reciclagem começou a partir de um projeto universitário, seguindo com base um dos temas dos ODS (Objetivos de Desenvolvimento Sustentável) 2030 da ONU( Organização das Nações Unidas).
O que nos motivou a criar um jogo foi poder realizar uma abordagem de gamificação na educação, utilizando elementos dos jogos em atividades educacionais, para que as pessoas possam aprendem melhor quando também estão se divertindo.

Software

Para o desenvolvimento foi utilizado o software Contruct 3

Programa

Os programas utilizados para o desenvolvimento do jogo foi o pixelart para a criação do personagem e do slime, para a ambientação do fundo do cenaria e para as construções foi utilizado sprites disponiveis nos sites: craftpix e itch.
